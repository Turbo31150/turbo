================================================================================
WORKFLOW (3 ETAPES)
================================================================================

ETAPE 1 : SCAN ET DETECTION DOUBLONS
  - Scan récursif du dossier
  - Calcul hash SHA-256 pour chaque fichier
  - Détection doublons (même contenu)
  - Barre de progression en temps réel

ETAPE 2 : ATTRIBUTION DES SCORES
  - Analyse de chaque fichier
  - Calcul score selon 6 critères
  - Score final : 0-100
  - Sauvegarde en base de données

ETAPE 3 : ORGANISATION
  - Score ≥60 : GARDER (structure organisée)
  - Score 30-59 : QUARANTINE (validation manuelle)
  - Score <30 : TRASH (poubelle)
  - Statistiques finales

================================================================================
EXEMPLES DE SCORING
================================================================================

FICHIERS SCORE ELEVE (80-100)
  - Image haute résolution (> 1 MB)         : 80
  - Document PDF récent (< 30 jours)       : 85
  - Code Python bien formaté               : 85
  - Vidéo récente                          : 75

FICHIERS SCORE MOYEN (30-59)
  - Image moyenne résolution               : 50
  - Document ancien (> 1 an)               : 40
  - Archive                                : 45

FICHIERS SCORE BAS (0-29)
  - Fichier vide                           : 0
  - Fichier .tmp                           : 10
  - Log ancien (> 7 jours)                 : 15
  - Doublon                                : 20
  - Fichier "backup_old.bak"               : 25

================================================================================
BASE DE DONNEES SQLITE
================================================================================

TABLES

1. files
   - path, name, size, extension
   - hash_sha256 (dédoublonnage)
   - score, score_reasons (JSON)
   - is_duplicate, duplicate_of
   - timestamps

2. decisions
   - file_id, action (MOVED/QUARANTINE/TRASH)
   - reason, executed, executed_at

REQUETES UTILES

  # Ouvrir la base
  sqlite3 disk_cleaner.db

  # Fichiers score bas
  SELECT name, score FROM files WHERE score < 30;

  # Doublons
  SELECT name, duplicate_of FROM files WHERE is_duplicate = 1;

  # Statistiques par extension
  SELECT extension, COUNT(*), AVG(score)
  FROM files
  GROUP BY extension;
