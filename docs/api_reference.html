GET /api/linux/health Sante complete du systeme

Retourne CPU, RAM, GPU (nvidia-smi), disque et nombre de services systemd actifs.

Reponse

{
  "status": "ok",
  "data": {
    "hostname": "m1-jarvis",
    "kernel": "6.17.0-19-generic",
    "cpu": { "cores": 16, "usage_pct": 12.3 },
    "ram": { "total_mb": 47104, "usage_pct": 42.1 },
    "gpus": [{ "name": "RTX 2060", "temp_c": 58 }],
    "disk": { "total_gb": 931.5, "usage_pct": 61.2 }
  }
}

Essayer

GET /api/linux/skills Liste de tous les skills

Retourne la liste complete des skills avec triggers, categorie et statistiques d'usage.

Reponse

{
  "data": {
    "count": 203,
    "skills": [{
      "name": "mode_gaming",
      "triggers": ["mode gaming", "lance les jeux"],
      "steps_count": 3,
      "usage_count": 12
    }]
  }
}
GET /api/linux/skills/{name} Detail d'un skill

Retourne le detail complet d'un skill : steps, triggers, categorie, stats.

Parametres

ParamTypeDescription
namepathNom du skill
POST /api/linux/skills/execute Executer un skill

Execute un skill par nom. Le skill est mis en file d'attente pour execution asynchrone.

Body (JSON)

{ "skill": "mode_gaming" }

Reponse

{
  "data": {
    "skill": "mode_gaming",
    "status": "queued",
    "steps": [{ "tool": "close_app", "args": {} }]
  }
}
GET /api/linux/voice/commands Commandes vocales

Liste des commandes vocales enregistrees, avec filtrage optionnel par categorie.

Query params

ParamTypeDescription
categorystringFiltrer par categorie (optionnel)
GET /api/linux/voice/corrections Corrections STT

Liste des corrections Speech-to-Text (wrong → correct) pour ameliorer la reconnaissance vocale.

{
  "data": {
    "count": 398,
    "corrections": [
      { "wrong": "jarvi", "correct": "jarvis" }
    ]
  }
}
GET /api/linux/voice/aliases Aliases sites & apps

Aliases des applications (APP_PATHS) et sites web (SITE_ALIASES) utilises par les commandes vocales.

GET /api/linux/voice/macros Macros vocales

Liste des macros : chaines de commandes executees en sequence.

GET /api/linux/brain/status Etat du brain

Etat complet du cerveau JARVIS : nombre de skills, patterns detectes, cycles d'apprentissage, metriques qualite.

{
  "data": {
    "total_skills": 203,
    "auto_learned": 45,
    "total_actions": 1250,
    "total_analyses": 89,
    "quality": { "avg_confidence": 0.78 }
  }
}
GET /api/linux/brain/predictions Predictions Markov

Suggestions predictives basees sur les patterns d'utilisation (Markov + heuristiques contextuelles).

{
  "data": {
    "suggestions": [{
      "reason": "morning_routine",
      "skills": ["status", "gpu"],
      "label": "Routine matinale"
    }],
    "detected_patterns": [{
      "sequence": ["gpu_status", "trading_scan"],
      "count": 7,
      "confidence": 0.85
    }]
  }
}
GET /api/linux/cluster/status Etat du cluster IA

Etat des noeuds LM Studio (M1/M2/M3) et Ollama (OL1) : modeles charges, GPU, VRAM.

{
  "data": {
    "lm_nodes": [{
      "name": "M1",
      "url": "http://127.0.0.1:1234",
      "gpus": 6,
      "vram_gb": 46,
      "default_model": "qwen3-8b"
    }],
    "ollama_nodes": [...]
  }
}
GET /api/linux/dominos Liste des dominos

Liste des dominos (learned actions) avec filtrage optionnel par categorie.

Query params

ParamTypeDescription
categorystringFiltrer par categorie : system, cluster, dev, trading, voice
POST /api/linux/dominos/execute Executer un domino

Execute un domino par ID. Accepte un ID numerique ou au format domino_XXX.

Body (JSON)

{ "id": "domino_42" }
// ou
{ "id": 42 }
GET /api/linux/profiles Profils vocaux

Liste des profils vocaux disponibles (builtin + custom).

{
  "data": {
    "profiles": [
      { "name": "default", "active": true },
      { "name": "dev", "active": false },
      { "name": "trading", "active": false }
    ]
  }
}
POST /api/linux/profiles/activate Activer un profil

Active un profil vocal. Persiste le choix dans data/active_profile.json.

Body (JSON)

{ "profile": "dev" }
GET /api/linux/notifications Historique notifications

Query params

Rapport du jour : actions executees, resume brain, GPU, charge systeme.
