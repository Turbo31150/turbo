UTILISATION BASIQUE
================================================================================

1. Démarrage system tray:
   python main.py

2. Ajout notification asynchrone:
   center = NotificationCenter()
   await center.add_notification(
       title="GPU Alert",
       message="Temp: 85°C",
       category="cluster",
       level="WARNING"
   )

3. Monitoring cluster:
   monitor = ClusterMonitor()
   monitor.update_from_ws({
       "gpu_temp": 75.0,
       "nodes_active": 5
   })
   print(monitor.get_tooltip())

4. Tests complets:
   python tests.py

5. Exemples intégration:
   python integration_example.py
