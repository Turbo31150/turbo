# Contributing

## DCO

By contributing, you certify your work is your own and you have the right to submit it under MIT License.

## Guidelines

**Python**: PEP 8, type hints, docstrings.

**JavaScript/TypeScript**: ESLint, Conventional commits.

**Commit convention**:
```
feat: add feature
fix: resolve bug
docs: update docs
chore: maintenance
```

## Issues

Open a GitHub issue with environment, steps to reproduce, expected vs actual behavior.
